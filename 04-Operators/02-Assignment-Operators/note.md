# Topic : assignment operators


# 1. What are assignment operators?
- Assignment operators are the operators that are used to assign values to variables. like =, +=, -=, *=, /=, %=, **=, ++, --.

## Assignment Operators
| Operator | Example | Equivalent |
|----------|---------|------------|
| `=`      | `x = 5` | `x = 5`    |  
| `+=`     | `x += 3`| `x = x + 3`|
| `-=`     | `x -= 2`| `x = x - 2`|     
| `*=`     | `x *= 4`| `x = x * 4`|
| `/=`     | `x /= 2`| `x = x / 2`|
|`%=`      | `x %= 2` | `x = x % 2`|
|`**=`     | `x **= 2`| `x = x ** 2`|
